# 웹사이트 링크를 넣으면 웹사이트의 모든 링크를 가져오는 터미널 도구

## 공식 문서같은 웹사이트에서 문서의 루트 링크를 넣으면 문서의 모든 링크를 가져오는 도구이다.

## 기능
1. rust로 개발됨.
2. 터미널에서 실행됨.
3. 웹사이트의 루트 링크를 넣으면 웹사이트의 모든 링크를 가져온다.

관련이 높지 않은 링크를 제외하기 위해, 그 페이지의 루트 링크의 하위 링크만 가져온다.
예를 들어, https://docs.rs/tokio/latest/tokio/ 에 루트 링크를 넣으면 https://docs.rs/tokio/latest/tokio/ 의 하위 링크만 가져온다.

터미널에서 명령어와 루트 링크를 입력하면 그 페이지의 루트 링크와 하위 링크들을 엔터 구분으로 출력한다.
명령어는 getlink이다.

## 예시
```bash
getlink https://docs.rs/tokio/latest/tokio/
```

## 기타 사항
- window, mac, linux에서 실행 가능하도록 하며, 개발이 편리하도록 개발 후 크로스플랫폼 배포를 자동화한다.
